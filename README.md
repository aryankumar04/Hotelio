# Hotelio

A modern hotel booking website to discover and explore amazing hotels.

## Features

- Browse featured hotels with beautiful photos
- View detailed hotel information including amenities, contact details, and location
- Photo gallery with fullscreen view
- Price calculator to estimate your stay cost
- Fully responsive design for all devices

## How to Use

1. Open `index.html` in your browser
2. Browse the featured hotels
3. Click on any hotel to see full details
4. Use the calculator to estimate your booking cost

## Customization

- **Add/Edit Hotels**: Modify `js/data.js` to update hotel information
- **Change Styling**: Edit `css/styles.css` to customize colors and design
- **Add Images**: Place your hotel images in the `assets/` folder
